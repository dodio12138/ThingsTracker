# 3) 第一版页面清单（移动优先）

## P0
1. 登录页 `/login`
2. 资产列表 `/assets`
3. 新增资产 `/assets/new`
4. 编辑/详情 `/assets/:id`

## P1
5. 参数模板管理 `/settings/params`

## 关键交互

### 资产列表卡片（简洁）
- 名称 + 型号
- 状态
- 购入价格
- 陪伴天数
- 日均成本

### 资产详情（分层）
- 第一层：基础字段（默认展开）
- 第二层：详细参数（Accordion 默认折叠）
  - 例如：性能参数、显示参数、影像参数、网络参数

### 新增资产流程
1. 填写基础字段
2. 按 `category` 自动加载对应 `asset_param_defs`
3. 填写参数值并保存到 `asset_param_values`
